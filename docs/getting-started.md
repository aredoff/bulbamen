# Getting started

## Requirements

- Node.js (LTS recommended)
- npm

## Install

```bash
npm install
# or
make install
```

## Development server

```bash
npm run dev
# or
make dev
```

Open **http://localhost:5173** (default Vite port).

## Production build

```bash
npm run build
# or
make build
```

Output: `dist/`

## Preview production build

```bash
npm run preview
# or
make preview
```

## Clean

```bash
make clean
```

Removes `dist/` and `node_modules/.vite`.

## Makefile

Run `make` or `make help` for a short list of targets.
